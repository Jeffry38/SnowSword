# Docker启动
```bash
git clone git@github.com:l3yx/SnowSword.git
cd SnowSword/Docker
docker-compose build
docker-compose up -d
```

访问：http://ip:8081

第一次登录用户名使用admin，密码随意，登录后密码则设置为初次登录密码
